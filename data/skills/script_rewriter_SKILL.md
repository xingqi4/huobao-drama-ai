# 剧本改写专家 — 专业技能指南

## 核心工作流程

1. **读取原文**：调用 `read_episode_script` 获取原始内容和已有剧本
2. **分析结构**：识别故事中的场景转换点、对白段落和动作描写
3. **逐场景改写**：按标准剧本格式改写每个场景
4. **保存结果**：调用 `save_script` 保存完整剧本

## 改写技巧

### 从叙述到视觉
- 小说中的心理描写 → 转化为可见的表情、动作、道具暗示
- "她心里很难过" → "[她的眼眶泛红，紧咬下唇，手指不自觉地攥紧了衣角]"
- "他很紧张" → "[他不自觉地搓着双手，目光闪烁不定]"

### 对白优化
- 删除冗余的称呼和语气词
- 增加潜台词层次
- 确保每个角色的语言风格有辨识度
- 对白应推动情节或揭示性格，避免纯信息传达

### 节奏把控
- 每个场景3-8个镜头段落为宜
- 高潮场景用更多短镜头
- 过渡场景用少量长镜头
- 对白密集场景可适当加入动作描写打破单调

### 场景标题规范
格式：内景/外景 - 具体地点 - 时间
- "内景 - 咖啡馆 - 日"
- "外景 - 城市街道 - 夜"
- "内景 - 医院病房 - 黄昏"

## 常见问题处理

- **原文过长**：优先保留核心情节，可适当精简次要描写
- **时间跳跃**：用转场说明标注，如"（三个月后）"
- **多线叙事**：用场景编号区分，确保每条线索连贯
- **闪回**：标注"（闪回）"和"（闪回结束）"
