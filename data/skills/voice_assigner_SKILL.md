# 音色分配师 — 专业技能指南

## 核心工作流程

1. **获取角色**：调用 `get_characters` 获取所有角色信息
2. **浏览音色**：调用 `list_available_voices` 获取可用音色列表
3. **匹配分配**：根据角色特征选择最合适的音色
4. **执行分配**：调用 `assign_voice` 为每个角色分配音色

## 音色匹配策略

### 按角色类型
| 角色类型 | 推荐音色特征 |
|---------|------------|
| 英雄主角 | 有力量感、辨识度高 |
| 反派 | 低沉、有压迫感 |
| 女主角 | 明亮、有感染力 |
| 智者/导师 | 沉稳、有阅历感 |
| 搞笑角色 | 活泼、有节奏变化 |
| 儿童 | 稚嫩、天真 |

### 按性格特征
- **外向活泼** → 明亮、节奏快
- **内向安静** → 柔和、节奏慢
- **冷酷理性** → 平稳、低沉
- **热情感性** → 温暖、有起伏

### 差异化原则
1. 主角必须使用独特音色，不可与其他角色共享
2. 重要配角之间应有明显区分
3. 性别不同的角色自然使用不同音色
4. 年龄差异大的角色使用不同音色
5. 小角色/龙套可以共用音色

## 分配注意事项

- 先为最重要的角色（主角、反派）分配最佳音色
- 确保同一对话场景中的角色音色有足够区分度
- 如果音色库不够用，优先保证主角的差异化
- 分配完成后，回顾检查是否有不合理的搭配
