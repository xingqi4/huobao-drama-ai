# 音色分配专家 — 专业技能指南

## 分配原则

1. **性别匹配**：男性角色用男声，女性角色用女声，不可混用
2. **年龄匹配**：少年/青年/中年/老年对应不同音色区间
3. **性格匹配**：音色气质要和角色性格一致
4. **角色定位**：主角用辨识度高的音色，配角用中性音色
5. **场景区分**：同一对话场景中的角色音色必须有足够区分度

## 核心工作流程

1. **获取角色**：调用 `get_characters` 获取所有角色信息（性别、年龄、性格）
2. **浏览音色**：调用 `list_voices` 获取可用音色列表及音色特征描述
3. **匹配分配**：根据角色特征选择最合适的音色
4. **执行分配**：调用 `assign_voice` 为每个角色分配音色
5. **汇总检查**：回顾分配结果，确保关键角色有差异化

## 音色匹配策略

### 按角色类型匹配

| 角色类型 | 推荐音色特征 | 典型感觉 |
|---------|------------|---------|
| 英雄主角 | 有力量感、辨识度高 | 正气、坚定、有穿透力 |
| 反派 | 低沉、有压迫感 | 阴冷、危险、控制欲 |
| 女主角 | 明亮、有感染力 | 清脆、温暖、有情感张力 |
| 智者/导师 | 沉稳、有阅历感 | 厚重、缓慢、自带故事感 |
| 搞笑角色 | 活泼、有节奏变化 | 夸张、跳跃、自带喜感 |
| 儿童/少年 | 稚嫩、天真 | 清亮、语速快、元气满满 |
| 旁白/解说 | 中性、沉稳 | 客观、有叙事感、不抢戏 |

### 按性格特征匹配

| 性格 | 音色方向 | 语速 | 语调 |
|-----|---------|------|------|
| 外向活泼 | 明亮、有弹性 | 快 | 上扬、起伏大 |
| 内向安静 | 柔和、低沉 | 慢 | 平缓、尾音轻 |
| 冷酷理性 | 平稳、低沉 | 中等 | 单调、没有感情波动 |
| 热情感性 | 温暖、饱满 | 中等偏快 | 丰富、带颤音 |

| 霸道强势 | 浑厚、有力 | 中等 | 果断、命令感 |
| 温柔体贴 | 甜美、柔和 | 慢 | 轻柔、尾音上扬 |

### 差异化原则（重要！）

1. **主角必须使用独特音色**，不可与其他角色共享
2. **重要配角之间应有明显区分**（同性别配角不能听起来像同一个人）
3. 性别不同的角色自然使用不同音色
4. 年龄差异大的角色使用不同音色
5. 小角色/龙套可以共用音色，但同一场景出现的龙套不能共用

### 分配优先级

按以下顺序分配，确保最重要的角色先拿到最佳音色：

1. 🔴 主角（男/女）→ 最独特、辨识度最高的音色
2. 🟠 反派 → 压迫感/阴冷感最强的音色
3. 🟡 核心配角 → 与主角有区分度的音色
4. 🟢 普通配角 → 中性音色
5. ⚪ 龙套/路人 → 可复用音色

## 分配注意事项

- 先为最重要的角色（主角、反派）分配最佳音色
- 确保同一对话场景中的角色音色有足够区分度
- 如果音色库不够用，优先保证主角的差异化
- 分配完成后，**必须回顾检查**：想象主角和反派对戏时，能听出区别吗？
- 旁白音色要选择不与任何主要角色撞音的中性音色
- 角色有年龄变化时（如回忆少年时期），可考虑分配两个音色

## 常见问题处理

| 问题 | 处理方法 |
|-----|---------|
| 音色库不够用 | 优先保证主角和反派差异化，配角可复用 |
| 两个角色特征相似 | 从语速、语调细微差异上区分（如一个慢一个快） |
| 角色性别不明 | 根据角色名和对白风格推断，无法确定时用中性音 |
| 反派需要"变脸" | 同一角色保持一个音色，通过语速和语调变化体现 |
| 音色列表看不懂 | 关注音色的性别、年龄、风格标签，与角色特征对齐 |

## 质量自检清单

- [ ] 主角是否使用了独特音色（未与其他角色共享）
- [ ] 同一对话场景中的角色音色是否可区分
- [ ] 角色性别与音色性别是否一致
- [ ] 角色年龄与音色年龄区间是否匹配
- [ ] 角色性格与音色气质是否协调
- [ ] 旁白音色是否不与主要角色撞音
